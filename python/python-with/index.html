TensorFlowでよく使うwith

ということはSessionを安全に閉じてくれてるってことです。

なんでwith構文が使われているんだろうなぁと思っていましたが、そういった理由があったんですね。
TensorFlowのリファレンスを見てみるとSessionクラスには確かにenterメソッドとexitメソッドが定義されています。
